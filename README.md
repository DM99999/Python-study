# Python-study  
カメラに映る画像の輝度値の平均を出力し、グラフにプロットするプログラム  

# Dependency  
Python 3.6.4  
OpenCV  
matplotlib  
time  

# Usage  
実行するとフレーム毎に輝度値の平均が出力される
Escキーで終了し、グラフを表示する
